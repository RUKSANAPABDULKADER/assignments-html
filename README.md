- `cd` - Change current directory.Ex:`cd Desktop`, `cd..`

- `ls` -List contents of a directory,Ex: `ls -a` lists hidden files as well.

- `pwd` -Display current working directory path

- `cat` -Display contents of a file.Ex:'cat README.md'
